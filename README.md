## Usage

Simply click on any district on the map to see:
- Population
- Number of dealers
- Total annual sales
- Monthly targets
- Current sales
- Achievement percentage
- Top dealer sales distribution

Toggle the "Color Grade" switch to visualize achievement levels across all districts.

## Technology Stack

- Pure HTML5, CSS3, and JavaScript
- SVG-based interactive map
- No external dependencies
- Static site ready for GitHub Pages

## Data

District insights include sales data, dealer information, and achievement metrics for all 14 districts of Kerala.
